## Spring 파일 한글 깨짐 해결 방법

1. 상단 메뉴 Windows > Preferences 클릭
2. Web > CSS Files, HTML Files, JSP Files 차례대로
3. Encoding 설정에서 'ISO 10646/Unicode(UTF-8)' 으로 변경  
   </br>
   < 참고 >
   - 브라우저 한글 깨짐 해결 방법 : ContentType UTF-8로 설정
